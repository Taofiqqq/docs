Ghana supports verification for mobile money wallets. You can resolve a mobile money number against a provider to retrieve the account holder's name.

| Attribute | Details |
|---|---|
| Account type | `mobile_money` |
| Currency | `GHS` |
| Name matching | Yes â€” returns account holder's full name |

## Mobile money verification

Send a `POST` request to `/core/accounts/resolve` with the following parameters:

### Required fields

| Field | Type | Description |
|---|---|---|
| `accountNumber` | string | The mobile money wallet number |
| `mobileMoneyCode` | string | The mobile money provider code (e.g. `MTN`) |
| `currency` | string | Must be `GHS` |
| `type` | string | Must be `mobile_money` |

### Request example

```json
{
  "accountNumber": "233591377522",
  "mobileMoneyCode": "MTN",
  "currency": "GHS",
  "type": "mobile_money"
}
```

### Successful verification (200)

```json
{
  "success": true,
  "message": "Account resolve successful",
  "data": {
    "_id": "67f7b231fc87184327317184",
    "accountName": "YVONNE EKOR KAKPA ONAH",
    "accountNumber": "233591377522",
    "bankCode": "MTN"
  }
}
```

### Failed verification (422)

```json
{
  "success": false,
  "error": "Account could not be resolved. Please check your selection and try again",
  "errorType": "UNPROCESSABLE_ENTITY"
}
```

## Result handling

| Response | Action |
|---|---|
| `success: true` | Account is valid. Compare `accountName` with your expected beneficiary name before proceeding with the payout. |
| `success: false` | Account could not be resolved. Do not proceed. Ask the customer to verify their account details and provider. |

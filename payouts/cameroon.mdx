## Overview

| | Mobile Money |
|---|---|
| **Currency** | XAF |
| **Scheme** | Local rails |
| **Settlement** | Instant |

Cameroon supports payouts to mobile money wallets only. Bank account payouts are not currently available.

## Mobile money payouts

Send XAF to mobile money wallets in Cameroon.

- **Payment destination:** `mobile_money_wallet`
- **Mobile money code required** — Pass the `mobileMoneyCode` for the recipient's provider. Retrieve available providers from the List Banks endpoint.
- **Phone number format** — Use country code + number without the `+` symbol (e.g., `237670123456`).

<Note>
  Bank account payouts to Cameroon are not supported at this time. Only mobile money is available.
</Note>

## Overview

| | Local NGN | USD (Domiciliary Account) |
|---|---|---|
| **Currency** | NGN | USD |
| **Method** | Bank account | Bank account |
| **Scheme** | Local rails | SWIFT |
| **Settlement** | Instant | 1-2 business days |

Nigeria supports two payout types: local NGN transfers to any Nigerian bank account, and USD payouts to domiciliary accounts at select banks via SWIFT.

## Local NGN payouts

Send NGN to any Nigerian bank account via local clearing rails. Payouts settle instantly.

- **Payment destination:** `bank_account`
- **No payment scheme required** — Fincra routes automatically via local rails.
- **Bank code required** — Pass the recipient's NIBSS bank code in the request. See your API reference for the full bank list.

## USD domiciliary account payouts

Send USD to Nigerian domiciliary (dom) accounts via SWIFT. Only the following 10 banks support USD dom account payouts:

| # | Bank |
|---|---|
| 1 | Access Bank |
| 2 | Fidelity Bank |
| 3 | First Bank |
| 4 | First City Monument Bank (FCMB) |
| 5 | Globus Bank |
| 6 | Polaris Bank |
| 7 | United Bank for Africa (UBA) |
| 8 | Union Bank |
| 9 | Wema Bank |
| 10 | Zenith Bank |

<Warning>
  USD payouts to Nigeria are only supported to domiciliary accounts at the banks listed above. Standard Naira accounts cannot receive USD.
</Warning>

- **Payment destination:** `bank_account`
- **Payment scheme:** `swift`
- **Sender details required** — International payouts require sender information for compliance.

## Overview

| | Mobile Money |
|---|---|
| **Currency** | XOF |
| **Scheme** | Local rails |
| **Settlement** | Instant |

Senegal supports payouts to mobile money wallets only. Bank account payouts are not currently available.

## Mobile money payouts

Send XOF to mobile money wallets in Senegal.

- **Payment destination:** `mobile_money_wallet`
- **Mobile money code required** — Pass the `mobileMoneyCode` for the recipient's provider. Retrieve available providers from the List Banks endpoint.
- **Phone number format** — Use country code + number without the `+` symbol (e.g., `221771234567`).

<Note>
  Bank account payouts to Senegal are not supported at this time. Only mobile money is available.
</Note>

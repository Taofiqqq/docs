## Create via dashboard

1. Log in to your [Fincra dashboard](https://app.fincra.com) and navigate to **Payments → Payment Links**.
2. Click **Create Payment Link**.
3. Fill in the link details — name, description, currency, amount, and fee bearer.
4. Click **Create**. Your shareable payment link is generated immediately.
5. Copy the link and share it with your customers via email, SMS, WhatsApp, or anywhere else.

## Create via API

Use the API to create payment links programmatically — useful when generating links at scale or integrating with your own tooling.

**Endpoint:** `POST /checkout-core/payment-links`

**Headers:**

| Header          | Description         |
| --------------- | ------------------- |
| `api-key`       | Your API secret key |
| `Content-Type`  | `application/json`  |

**Request fields:**

| Field                    | Type    | Required | Description                                                                                      |
| ------------------------ | ------- | -------- | ------------------------------------------------------------------------------------------------ |
| `name`                   | string  | Yes      | A label for the payment link. Shown to customers on the pre-checkout page.                       |
| `description`            | string  | Yes      | A description of what the customer is paying for.                                                |
| `currency`               | string  | Yes      | The currency for the payment link. See [Supported Currencies](/payments/supported-currencies).   |
| `settlementDestination`  | string  | Yes      | Where funds settle. Currently only `wallet` is supported.                                        |
| `amount`                 | number  | No       | The payment amount. If not fixed, customers can enter their own amount on the pre-checkout page. |
| `feeBearer`              | string  | No       | Who bears the transaction fee. `business` (default) or `customer`.                              |
| `redirectUrl`            | string  | No       | URL to redirect the customer after a successful payment.                                         |
| `successMessage`         | string  | No       | Custom message shown to the customer after successful payment.                                   |
| `metadata`               | object  | No       | Additional data to attach to the payment link.                                                   |

**Example request:**

```bash
curl --request POST \
  --url https://api.fincra.com/checkout-core/payment-links \
  --header 'api-key: <YOUR_SECRET_KEY>' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Taofiq Aiyelabegan",
    "description": "Pay for goods",
    "currency": "NGN",
    "settlementDestination": "wallet",
    "amount": 100,
    "feeBearer": "customer"
  }'
```

**Example response:**

```json
{
  "status": true,
  "message": "Payment link created successfully",
  "data": {
    "id": 1213,
    "name": "Taofiq Aiyelabegan",
    "description": "Pay for goods",
    "currency": "NGN",
    "settlementDestination": "wallet",
    "amount": 100,
    "feeBearer": "customer",
    "reference": "c0f00954df6c34d992a20",
    "businessId": "67d8360649685a0012574974",
    "redirectUrl": null,
    "successMessage": null,
    "isAmountFixed": false,
    "disabled": false,
    "createdAt": "2026-02-19T09:26:36.871Z",
    "updatedAt": "2026-02-19T09:26:36.871Z"
  }
}
```

<Note>
  The API response does not include a shareable URL. To get the payment link URL, go to **Payments → Payment Links** in your Fincra dashboard and find the link by name or reference.
</Note>

## Retrieve a payment link

Fetch the details of a single payment link by its ID.

**Endpoint:** `GET /checkout-core/payment-links/:id`

```bash
curl --request GET \
  --url https://api.fincra.com/checkout-core/payment-links/1213 \
  --header 'api-key: <YOUR_SECRET_KEY>'
```

## List all payment links

Retrieve all payment links created for your business.

**Endpoint:** `GET /checkout-core/payment-links`

```bash
curl --request GET \
  --url https://api.fincra.com/checkout-core/payment-links \
  --header 'api-key: <YOUR_SECRET_KEY>'
```

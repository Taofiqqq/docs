Multi-currency virtual accounts let you receive international and cross-border payments via bank transfer. Like NGN virtual accounts, funds received settle into your Fincra wallet in the corresponding currency.

Multi-currency virtual accounts are **permanent only** — there are no temporary accounts for non-NGN currencies.

## Supported currencies

| Currency | Scheme | Notes |
|---|---|---|
| USD | ACH, SWIFT | SWIFT accepts transfers from anywhere in the world, including Nigerian domiciliary accounts |
| EUR | SEPA, SEPA Instant | For Eurozone transfers |
| GBP | FPS, CHAPS | For UK transfers |
| KES | KEPS | For Kenyan bank transfers |
| TZS | — | For Tanzanian bank transfers |

## Create a multi-currency virtual account

All multi-currency permanent VAs use the same endpoint — the payload varies by currency.

**Endpoint:** `POST /profile/virtual-accounts/requests`

**Headers:**

| Header | Description |
|---|---|
| `api-key` | Your API secret key |
| `Content-Type` | `application/json` |

## Receive payments

When funds arrive in a multi-currency virtual account, Fincra sends a `charge.successful` webhook. The funds settle into your Fincra wallet in the same currency as the virtual account.

For full webhook setup, see [Webhooks](/webhooks/overview).

## RFI — Request for Information (USD, EUR, GBP)

For incoming transfers in USD, EUR, or GBP, Fincra's compliance team may place funds on hold and request supporting documentation before releasing them. This is a standard compliance requirement for international wire transfers.

**When it happens:**

Fincra compliance will notify you by email when an incoming transfer is held for review. The email will specify what documents are required.

**What may be requested:**
- Source of funds documentation
- Invoice or contract related to the transfer
- Other compliance documentation as required by the transfer details

**What to do:**

Respond promptly to the compliance email with the requested documents. Delays in responding will extend the hold period.

## Retrieve multi-currency virtual accounts

**Get a single virtual account by ID:**

```bash
curl --request GET \
  --url https://api.fincra.com/profile/virtual-accounts/<virtual_account_id> \
  --header 'api-key: <YOUR_SECRET_KEY>'
```

**List by currency:**

```bash
curl --request GET \
  --url https://api.fincra.com/profile/virtual-accounts?currency=USD \
  --header 'api-key: <YOUR_SECRET_KEY>'
```

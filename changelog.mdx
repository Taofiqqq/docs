<Update label="March 2026">
## Introducing the New Fincra Documentation

We've completely rebuilt Fincra's developer documentation
from the ground up.

**What's new:**

- New docs platform with improved navigation and search
- Restructured around products: Payments, Payouts,
  Transactional FX, Direct Debit, and Verification
- Clear getting started guides for each product
- Comprehensive webhook documentation
- Server-side and mobile SDK references
- Sandbox testing guides with test credentials

**What's changed:**

- "Pay-Ins/Collections" → Payments
- "Payouts (Withdrawal)" → Payouts
- "Conversions" → Transactional FX
- "Identity Management" → Verification
- "Checkout Redirect" → Hosted Payment Page
- "Checkout Standard" → Drop-in Checkout

</Update>
